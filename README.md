# 📊 Advanced Area Plot with Legend & Grid
### 🚀 Project Overview
This project presents a clean and modern way to visualize comparative sales data using an area plot. Instead of plain numbers, it transforms data into an intuitive visual format that highlights trends, differences, and growth over time.
It’s designed with simplicity in mind, making it perfect for beginners while still useful as a base for more advanced data visualization projects.

### 🎯 Key Highlights

✔ Compare multiple datasets visually

✔ Clean shaded area representation

✔ Built-in legend for clarity

✔ Grid system for better readability

✔ Lightweight and easy to understand

### ⚙️ How It Works

The project uses simple lists to simulate monthly sales data:

* x → Months (time axis)

* sales_2024 → Sales data for 2024

* sales_2025 → Sales data for 2025

Using fill_between(), the program shades the area under each dataset, allowing easy comparison between the two years.

### 📊 Output Description
- The generated plot will include:

- Two shaded regions representing yearly sales

- A legend to differentiate datasets

- A grid for better data alignment

- Clearly labeled axes and title
